# DankCaffeine

A tiny [DankMaterialShell](https://danklinux.com) plugin that prevents your screen from going idle

## Features

- **DankBar widget** — coffee icon in your bar, click to toggle. Highlights when active.
- **Control Center toggle** — standard tile with status text.

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| Caffeine Active | Off | Current toggle state |
| Restore on startup | On | Re-enable caffeine automatically if it was active when DMS last closed |

## Compatibility

- **Compositors:** Any (niri, Hyprland, Sway, labwc, etc.)
- **Distros:** Any with systemd
- **Requires:** DMS >= 1.2.0
